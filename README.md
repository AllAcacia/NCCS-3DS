# NetCORE Connection Service (NCCS-3DS)

--- 

## About
NCCS-3DS (NetCORE Connection Service) is a Homebrew implementation of a reliable transport-layer service for the 3DS, using the provided UDS services. It takes inspiration from TCP but primarily for connection / acknowledgement / ordered / window oriented schemes. I am also working on a Club Penguin Card-Jitsu recreation, so if you'd like to join my community Discord, the server link is here: https://discord.gg/2aw3GXCU8D.
